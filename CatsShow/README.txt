
This code is based on MVVM architecture and support followings:

. Localization
. Development and Production Schemes
. Caching of the images using Kingfisher Image Iibrary
. Few Unit Test Cases 


How to compile : 

As this code is having a sample code to support caching of images using Kingfisher image library hence go to the XCode's menu and select File->Packages->Resolve Package Version, so that Kingfisher framework can be downloaded.
